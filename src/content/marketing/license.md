# Licence Terms — Asteris Affiliates

Asteris Affiliates is GPL-2.0+ software. All tiers — Free, Starter, Pro, Agency, and Founder — use the same open-source licence.

---

## GPL-2.0+ licence

The Asteris Affiliates plugin code is distributed under the GNU General Public License version 2.0 or later. This means:

- You can use the software on any site within your tier's site limit
- You can modify the code for your own use
- If you distribute modified versions, you must do so under GPL-2.0+
- The licence does not restrict your commercial use of the software (you can use Asteris Affiliates on a client's commercial store)

The GPL licence covers the plugin software. It does not cover access to updates, support, or future modules — those are commercial services attached to your annual or lifetime subscription.

---

## Site limits per tier

| Tier | Sites |
|------|------:|
| Asteris Affiliates Free | Unlimited (GPL — no restriction) |
| Asteris Starter | 1 production site |
| Asteris Pro | 3 production sites |
| Asteris Agency | Unlimited production sites |
| Founder | 1 production site |

**What counts as a site:** A production site is a publicly accessible domain where Asteris Affiliates serves real traffic. A local development environment working on that same project doesn't count as a separate site.

**Staging sites:** A staging subdomain for the same production site (e.g. staging.yourdomain.com) counts as part of the same site licence.

**Multisite networks:** Asteris Affiliates is not officially supported on WordPress multisite at v1.0. Using a single-site licence on a multisite installation is outside the supported scope.

## Transferring a licence

**Starter, Pro, Agency:**
You can move a licence to a different domain within your site limit. Email support@ with the old and new domain. No fee. Processed within 1 business day.

**Founder:**
Transferable to a different domain within the same buyer's ownership. Email support@. No fee. No limit on transfers.

Not transferable to a different person or business entity. A licence sold or given away is a licence violation. We may revoke licences found to be transferred to third parties.

## When an annual licence lapses

If you don't renew, your Asteris Affiliates licence enters a lapsed state:

- Asteris Affiliates continues to function on your site (the GPL code doesn't disappear)
- You stop receiving plugin updates
- You lose access to the customer download portal
- Your support access ends

To resume updates and support, renew via the customer portal. Renewals are pro-rated from the date you renew (not backdated to the original expiry).

---

## Founder — duration

Founder is valid for as long as My Cosmic Message Pty Ltd t/a Asteris Commerce operates. If Asteris Commerce were to cease operations, the GPL licence means you retain the right to continue using the Asteris Affiliates version you last downloaded. New modules, updates, and support would also cease.

We have no plans to cease operating. This clause is included for transparency.

## Developer rights

As a developer using Asteris Affiliates on client sites:

- **Asteris Free:** No restriction. GPL applies. Use it on any client site.
- **Asteris Pro (3 sites):** Install on up to 3 client sites under one licence.
- **Asteris Agency (unlimited):** Install on as many client sites as you maintain. The Agency licence doesn't extend to clients who self-manage the plugin after a build is complete — they would need their own licence.
- **Reselling:** You cannot resell Asteris Affiliates licences or bundle them into your own product offering.
