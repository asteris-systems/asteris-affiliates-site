# Migrate to Asteris Affiliates

Asteris Affiliates replaces up to 19 separate plugins. If you're switching from one or more of them, migration guides are below.

Most stores complete a migration in under an hour. The guides cover data export from your current plugin, import into Asteris Affiliates, and verification steps.

---

## Available migration guides

**[From Yoast SEO →](/migrate/from-yoast)**
Covers meta titles, descriptions, redirects, breadcrumb settings, and schema. The migration preserves your existing SEO data.

**[From Rank Math →](/migrate/from-rank-math)**
Covers meta data, schema, redirects, and 404 monitor data. Rank Math has a built-in export that Asteris Affiliates imports directly.

**[From YITH Wishlist →](/migrate/from-yith)**
Covers saved wishlists, customer wishlist data, and shortcode replacement.

**[From WP Overnight PDF Invoices →](/migrate/from-wp-overnight)**
WP Overnight was acquired by UpdraftPlus. Covers template migration and order history PDF regeneration.

**[From Pretty Links →](/migrate/from-pretty-links)**
Covers link export, redirect preservation, and click history migration.

---

## Before you migrate

Install Asteris Affiliates on a staging site first. Run the migration there. Verify everything works. Then run it on production.

If anything breaks, email support@asterisaffiliates.com. First response within 1 business day for paid tiers.
