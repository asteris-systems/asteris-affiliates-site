## Subscribing to release notifications

You'll be notified about every release via:

- **Email** — sent to your purchase email by Lemon Squeezy on every release, with a summary + the changelog link
- **WordPress admin** — the plugin update notice + the changelog excerpt show in **Plugins → Updates** when an update is available
- **Friday built-in-public post** — short weekly summary of what shipped + what's coming next, lands at `/blog` (when live)

If you don't want release emails, you can unsubscribe from the Lemon Squeezy email at any time — your plugin will still auto-update via WordPress's standard update mechanism.

## When something breaks

If a release breaks your site, see [Troubleshooting → "A plugin update broke my site"](/docs/troubleshooting#a-plugin-update-broke-my-site---how-do-i-roll-back) for the rollback path. We treat rollback reports as P1 bug reports and patch within your tier's support SLA.
