The AI swipe-copy library gives affiliates a starting point for promoting your products. v1.1.0 seeds **16 pre-written snippets** out of the box; affiliates can generate AI-personalised variants via the optional Asteris AI module (from Asteris for WordPress).

## What's in the seed library

16 snippets covering:

- **Email subject lines** (4)
- **Email body templates** (4)
- **Tweet/X templates** (3)
- **LinkedIn templates** (3)
- **Blog post intros** (2)

Each snippet has placeholders (`{product_name}`, `{affiliate_link}`, `{discount_code}`) that the affiliate fills in.

## How affiliates use it

1. Log in to the affiliate portal
2. **Swipe copy → Library**
3. Browse snippets, filter by channel (email / social / blog)
4. Click **Copy** → snippet copied to clipboard with their referral link auto-substituted

## Admin: customise the library

**Affiliates → Swipe copy → Snippets**:

- Edit existing seed snippets to match your brand voice
- Add custom snippets per product or campaign
- Tag snippets by category
- Disable snippets you don't want affiliates using

## AI variants (optional)

If you have the Asteris AI module installed (from Asteris for WordPress), affiliates can click **Generate AI variant** on any snippet. The plugin sends the snippet + product context to the AI provider and returns a re-written version.

Without the AI module, affiliates only see the static library.

## Cost (with AI)

AI calls use **your** Asteris AI module's configured provider (OpenAI, Anthropic, etc.). Typical cost: $0.001 – $0.005 per generation.

Set a per-affiliate monthly cap to avoid runaway costs: **Settings → AI swipe → Per-affiliate monthly generation cap → 50** (default).

## Per-affiliate snippet performance

Track which snippets convert best. **Reports → Swipe copy** shows clicks/conversions per snippet across all affiliates. Use to identify top performers and double down.

## Privacy

AI generations are not logged with personally identifiable info — only the snippet text + product slug is sent to the AI provider.
